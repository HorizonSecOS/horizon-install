# Horizon-Install

**Horizon-Install** is the official installation script for **HorizonSec**, an Arch-based penetration testing Linux distribution built for both beginners and seasoned security professionals. This script allows you to install HorizonSec using either the provided **HorizonSec Install ISO** or any standard **Arch Linux ISO**.

---

## üöÄ Features

* **Beginner-friendly** guided installation
* **Advanced options** for experienced users
* Supports installation from:

  * The official HorizonSec Install ISO (recommended)
  * A standard Arch Linux ISO
* Configures all core HorizonSec tools, environment, and optimizations
* Automates setup for:

  * Kernel and system configuration
  * Desktop environment (if chosen)
  * Security toolchain installation
  * Networking and drivers
  * User accounts and hardening
